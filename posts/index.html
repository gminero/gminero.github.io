Lwc Aura Wrappers

In this post, we will create an Aura wrapper component around a LWC that fires an APP event to an external Aura component. Getting Started You will need to get yourself setup with Salesforce-DX, if you haven’t had the chance to do so, this is a good time to get your environment setup since we will use it to deploy our LWC to a scratch org for testing.
